# Flavor 엔진 학술 리서치 종합 (2026-03-12)

## 1. 십신-MBTI 매핑

### 핵심 논문: 사주명리학 8가지 성격유형과 MBTI 상관연구
- 대상: 성인 170명, MBTI GA검사지 + 사주간명지
- **유의미한 매핑 발견**:
  - 상관(傷官) ↔ 외향감정(EF)
  - 정관(正官) ↔ 외향사고(ET)
  - 정재(正財) ↔ 내향감각(IS)
  - 편재(偏財) ↔ 외향감각(ES)
  - 식신(食神) ↔ 내향감정(IF)
  - **편인, 정인은 MBTI와 무관** → 독자 모델 필요
- 출처: https://scienceon.kisti.re.kr/srch/selectPORSrchArticle.do?cn=DIKO0012741594

### 보조 논문
- MBTI 외향(E) ↔ 목/화 비율, 내향(I) ↔ 금/수 비율 (KCI 등재)
- 성인 250명 대상 사주-MBTI 상관분석 (DIKO0013046544)

## 3. BaziQA Dataset

### BaziQA-Benchmark (arXiv 2602.12889, 2026-02)
- Global Fortune-teller Competition(2021-2025) 문제 200개 기반
- 완전한 명반 계산 포함 (천간/지지, 십신, 대운, 세운)
- **GitHub 공개**: https://github.com/ChenJiangxi/BaziQA
- LLM 평가: 우연 이상이지만 포화에는 한참 못 미침

### BaZi Character Simulation (arXiv 2510.23337, 2025-10)
- BaZi 기반 페르소나 추론 QA 데이터셋
- BaZi 강화 모델 → 30.3~62.6% 정확도 향상

## 4. TCM 오행-심리학

### R-5EPQ (SSRN 2023)
- 15문항 오행 성격 설문지
- **오행은 Big Five와 독립적 구인** — 별도의 성격 차원 측정
- 오행을 Big Five의 대체가 아닌 **보완 차원**으로 활용 가능

### 오행별 성격-취향 프로파일
| 오행 | 핵심 성격 | 취향 방향 |
|------|-----------|-----------|
| 목 | 성장지향, 추진력 | 자기계발, 아웃도어, 새로움 |
| 화 | 카리스마, 감각 추구 | 비비드, 페스티벌, 매운 음식 |
| 토 | 조화/안정, 충성 | 전통, 뉴트럴, 한식 |
| 금 | 질서, 완벽주의 | 미니멀, 클래식, 정돈 |
| 수 | 유연, 내면 탐구 | 명상, 독서, 실험적 경험 |

## 5. 십신→취향 직접 매핑 (학술 근거 약함, 실무자 레벨)

| 십신 | 아키타입 | 취향 방향 |
|------|----------|-----------|
| 비견 | 자기동일 | 브랜드 충성, 독립적 취향 |
| 겁재 | 경쟁자 | 트렌드 민감, 과시적 소비, 한정판 |
| 식신 | 미식가 | 미식, 감성 경험, 예술, 편안한 패션 |
| 상관 | 반항아 | 실험적, 서브컬처, 독특한 스타일 |
| 편재 | 모험가 | 다양한 경험, 여행, 새로운 음식 |
| 정재 | 관리자 | 가성비, 실용, 전통 |
| 편관 | 투사 | 강렬한 경험, 스포츠, 자극적 음식 |
| 정관 | 관리직 | 클래식, 정돈, 브랜드 정통성 |
| 편인 | 탐구자 | 마이너/비주류, 영적 경험, 독서 |
| 정인 | 스승 | 교육 콘텐츠, 전통문화, 건강식 |

## 엔진 적용 권장사항

1. **BaziQA GitHub 참조** — 십신 계산 로직 + 데이터 구조
2. **2계층 설계**: 오행 = 거시적 성향, 십신 = 세부 취향 방향
3. **MBTI 매핑은 마케팅용으로만** (편인/정인 설명 불가)
4. **초기: 이론 기반 매핑 → 데이터 축적 후 실증 보정**
